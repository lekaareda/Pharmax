# Pharmax Flutter Starter

هذا مشروع Flutter مبدئي لتطبيق صيدلية إلكترونية باسم **Pharmax**.

## المتطلبات

- Flutter SDK (3.22 أو أحدث)

## التشغيل

```bash
flutter pub get
flutter run
```

## الاختبارات

```bash
flutter test
```

## ماذا يحتوي الآن؟

- شاشة رئيسية فيها:
  - بحث عن الأدوية
  - أقسام رئيسية
  - منتجات شائعة
  - شريط تنقل سفلي

> يمكنك تعديل الألوان والنصوص وربط البيانات الفعلية بسهولة من `lib/main.dart`.
